# Real-Time-Trend-Analyzer
A Real-Time News Sentiment &amp; Trend Analysis Dashboard built using Python, NLP, Streamlit, VADER, and Google News RSS Feed. The system automatically fetches live news headlines, analyzes their sentiment (Positive / Neutral / Negative), classifies them into categories (Politics, Tech, Sports, etc.), and visualizes the results with expressive charts.
